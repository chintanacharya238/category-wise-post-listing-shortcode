== Description ==

**Important Note:**
<p>A simple shortcode to list a taxonomy and the posts for each term ex: <b>[cat_listting post_type="post" taxonomy="category" slug="blog"]</b>. </p>
<p>For Set Category Listing Design change design in template/category_listing.php this file.</p>


== Installation ==

1. Upload `category-wise-post-listing-shortcode` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Just Add Shortcode [cat_listting post_type="post" taxonomy="category" slug="blog"].
